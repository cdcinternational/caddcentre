Mobile Version (1 image per slide) | Desktop Version (2 images per slide)

Responsive Carousel Features

  • Mobile Version: Shows 1 image per slide with optimized mobile images
  • Desktop Version: Shows 2 images per slide with desktop images
  • Auto Play: Carousel automatically advances every 5 seconds
  • Pause on Hover: Auto-play pauses when you hover over the carousel
  • Navigation Controls: Use the arrow buttons to manually navigate
  • Indicators: Click on the dots to jump to specific slides
  • Continuous Loop: Carousel wraps around from last to first slide

Implementation Details

  • Uses Bootstrap's responsive classes (d-none d-md-block and d-md-none)
  • Separate carousel instances for mobile and desktop
  • Different images optimized for each device type
  • Auto-play functionality with 5-second interval
