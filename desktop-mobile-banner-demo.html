Banner Comparison

Feature Desktop Banners Mobile Banners
Display Full width, 500px height Full width, 300px height
Image Optimization High resolution for large screens Optimized for smaller screens
Content Focus Detailed information and visuals Simplified content and clear CTAs
File Size Larger files (100KB+) Smaller files for faster loading
Aspect Ratio Landscape (16:9 or similar) More square or portrait

Implementation Guide

HTML Structure

The carousel uses Bootstrap's responsive classes to show different banners:

Desktop Version:
<div class="carousel-inner d-none d-md-block">
  <div class="carousel-item active">
    <img src="desktop-banner.jpg" alt="Desktop Banner">
  </div>
</div>
Mobile Version:
<div class="carousel-inner d-md-none">
  <div class="carousel-item active">
    <img src="mobile-banner.jpg" alt="Mobile Banner">
  </div>
</div>
Key Classes
  • d-none d-md-block - Hidden on mobile, visible on desktop
  • d-md-none - Visible on mobile, hidden on desktop
  • object-fit: cover - Maintains aspect ratio while filling container
  • w-100 - Makes image full width
Best Practices
  • Use different images optimized for each screen size
  • Keep mobile banners simpler with larger text
  • Ensure fast loading times for mobile versions
  • Test on actual devices for proper display
  • Use descriptive alt text for accessibility
